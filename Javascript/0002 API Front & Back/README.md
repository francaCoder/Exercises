# Consumir API

## API

Criar uma API
    * Iniciar o projeto com `npm init -y`
    * Instalar as dependências `npm i express nodemon cors axios`
    * Criar uma rota de acesso para a API
    * Servir a API
    * Habilitar o CORS
    * HAbilitar o nodemon para reiniciar o server
